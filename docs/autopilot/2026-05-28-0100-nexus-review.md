## Connector type

ActionBridge webhook signing secret-management connector using server-side Google Secret Manager REST resolution.

## Supported actions

- `read`: production readiness/configuration-shape preflight.
- `read`: live access probe with redacted audit summary.
- `transactional`: resolve HMAC signing secret server-side before webhook delivery/rotation.

## Required auth/session

Server-side only. Requires `ACTIONBRIDGE_SECRET_MANAGER_PROVIDER=google_secret_manager_rest`, `ACTIONBRIDGE_SECRET_MANAGER_REQUIRED=true`, project id, and access token/service identity. No secrets are exposed to browser/client/agent prompts.

## Risk per action

- Preflight: low; reports config shape only.
- Live access probe: medium; fails closed and returns digest-only/redacted summaries.
- Secret resolution: high; acceptable for pilot path because it is server-only, bounded-timeout, redacted, and blocks unresolved refs before network delivery.
